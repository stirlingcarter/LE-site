LE

LE is a smartphone app for musicians to improve their playing speed and vocabulary. LE uses math to be very efficient.

Explanation

LE works by obsessively measuring timings. It builds a rich relational model of competencies across a vocabulary.

LE is completely flexible. It can be used to target arbitrarily broad or narrow areas of performance.

Concrete usage examples:

  • Learn each note on the guitar fretboard
  • Improve retrieval to below 300ms for every major 7 chord inversion in L and R hands on the piano
  • Optimize speed of retrieval for the chords in a specific song

LE is a very fast and easy way to reach goals like these.
